# Insurance Claims Analysis Dashboard

## Project Overview
This project is an Excel-based dashboard analyzing insurance claims data. The goal was to summarize claim volume, total claim costs, average claim amounts, and claim trends across different categories such as claim type, status, region, and time period.
The dashboard was built using Excel PivotTables, PivotCharts, slicers, KPI cards, formulas, and data cleaning techniques.

## Business Objective
Insurance companies need to monitor claim activity to identify cost drivers, claim trends, and areas of operational risk. This dashboard provides a high-level view of claims performance and allows users to filter the data interactively.

## Tools Used
- Microsoft Excel
- PivotTables
- PivotCharts
- Slicers
- Formulas
- Data cleaning and formatting

## Key Features
- Interactive KPI cards for total claims, total claim amount, average claim amount, and claim status
- PivotCharts showing claim trends and category breakdowns
- Slicers for filtering by region, claim type, and claim status
- Cleaned and formatted raw claims data
- Data dictionary explaining the fields used in the analysis

## Key Insights
- Identified major claim cost drivers by category and region
- Compared claim counts and claim amounts across different claim statuses
- Created an interactive dashboard for quick claims performance review
